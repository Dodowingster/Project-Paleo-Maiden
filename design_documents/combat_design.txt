Battle System:
1) Character Stats:
- HP
- ATK
- DEF
- SPD
- STA (min and max)
- Action Goal limit
- Action Stocks
- fwd move spd mult
- bkwd move spd mult
- fwd dash spd mult
- min fwd dash duration
- max fwd dash duration
- bkwd dash spd mult
- min bkwd dash duration
- max bkwd dash duration

2) Character differences (besides stats):
- different base attack
    - attacks determine threat range
    - ??? with more techniques, threat range is determined by the longest range attack thats ready for use (important!)
- different strategies

3) Actions:
- can gain action goal:
    - walk forward
        - can be interrupted with attack/block/clash anytime
    - walk backward
        - can be interrupted with attack/block/clash anytime
    - dash forward
        - has min (and maybe max) duration
        - can only be interrupted with attack/block/clash after min duration
    - dash backward
        - has min (and maybe max) duration
        - can only be interrupted with attack/block/clash after min duration
- cannot gain action goal:
    - attack
        - consumes an action stock and reset action goal
    - block
        - consumes an action stock? resets action goal?
    - clashing
        - clash winner gains action goal
        - clash loser's action goal is stuck in place until actionable

4) Situations:
- 2 attacks hitting at the same time:
    - higher base damage wins
    - if same base damage, higher action goal wins
