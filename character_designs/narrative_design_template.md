## Notes for Character Designers

- **Core Conflict First**: Start with Section 1's core conflict. Everything else flows from this.
- **Show, Don't Tell**: Use scenes and dialogue to reveal character rather than exposition.
- **Opponent Selection**: Each opponent should represent a different facet or challenge to the character's growth.
- **Callback Structure**: Reference the formative story in later scenes to show how the character has evolved.
- **Affinity Event**: This should be deeply personal and reveal something only the Lecturer has earned through trust.
- **True Ending**: Connect this to the earliest story elements for narrative satisfaction.
